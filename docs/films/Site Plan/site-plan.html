Overview

Purpose

The purpose of this site is to view and rate films to see which ones people like and dislike. This site will help to guide them to movies that are top rated and will bring the most joy.

Audience

The audience for this site is anyone who loves movies and is looking for something good to watch.

Dynamic elements

I would like to add a search function to search for the titles of different films. Similar to the recipe book site, I would also like to use javascript to create "cards" for each film with all the information about it. When the page is loaded, a random film card would be featured.
